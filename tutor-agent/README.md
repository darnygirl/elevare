# Elevare AI Tutor Agent

An intelligent tutoring system for speech, confidence, and communication courses.

## Overview

The AI Tutor Agent provides personalized coaching across three Elevare programs:
- **Lingua** - Language learning (15 languages)
- **Rise** - Confidence building  
- **Flex Nexus** - Professional communication

## AI Tutor Capabilities

### Natural Conversation
- Engage in dialogue about course content
- Answer questions naturally
- Adapt teaching style to student responses

### Speech Analysis
- Transcribe student speech
- Identify pronunciation issues
- Analyze pace, rhythm, filler words
- Provide real-time feedback

### Personalized Learning
- Track student progress
- Identify weak areas
- Adapt difficulty
- Generate personalized practice

### Practice Sessions
- Role-play conversations
- Debates and discussions
- Presentations with feedback
- Interview simulations

## Usage

```bash
# Install dependencies
pip install openai python-dotenv

# Set OpenAI API key
export OPENAI_API_KEY='your-key'

# Run the tutor
python tutor.py
```

## AI vs Human Comparison Framework

### Metrics Tracked

| Metric | AI Tutor | Human Tutor |
|--------|----------|-------------|
| Response Time | Instant | Minutes-Hours |
| Availability | 24/7 | Scheduled |
| Cost per Session | ~$0.01 | $15-75 |
| Patience Level | Infinite | Varies |
| Personalization | Data-driven | Intuitive |
| Emotional Support | Limited | Strong |
| Cultural Context | Learning | Natural |
| Practice Conversations | Unlimited | Limited |

### Study Design

```
Group A: AI Only
- All learning through AI Tutor
- Unlimited practice sessions
- Self-paced

Group B: Human Only  
- All learning through human coaches
- Scheduled sessions
- Structured curriculum

Group C: Hybrid
- AI for practice & feedback
- Human for emotional support & complex topics
- Scheduled check-ins

Group D: Control
- Traditional self-study
- No personalized feedback
```

### Assessment Methods
- Pre/post confidence surveys
- Speech clarity scores
- Grammar accuracy tests
- Student satisfaction surveys
- Cost per improvement metric
