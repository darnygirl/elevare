## Edit 1 — Fix the $199 vs "two hundred" mismatch

**Why:** site checkout shows `$199/mo`. Pitch says "two hundred a month." Customer hears one number, sees another → trust crack at the moment of decision.

### File: `elevare-pitch-general-2min.docx`

**FIND:**
```
Two hundred a month if you want a human tutor dropping in.
```

**REPLACE:**
```
One ninety-nine a month if you want a human tutor dropping in.
```

## Edit 2 — Remove "WhatsApp scenarios" from General tier description

**Why:** WhatsApp drills are the $29 AI Practice Bot **add-on**, not part of the $99 General tier. Bundling them in the pitch devalues the add-on and contradicts the site.

### File: `elevare-pitch-general-5min.docx`

**FIND:**
```
Ninety-nine dollars a month, self-serve — groups + AI facilitator + WhatsApp scenarios.
```

**REPLACE:**
```
Ninety-nine dollars a month, self-serve — small groups + AI facilitator + named feedback after every session.
```

(Note: this also picks up Edit 1 if you do them in order — just make sure to also catch the "Two hundred" → "One ninety-nine" swap on the next sentence.)
