## 3. GOOGLE DRIVE FOR UPLOADS

### Create Folder for Tutor Resources

1. Go to https://drive.google.com/
2. Click "+ New" → "Folder"
3. Name: "Elevare Tutor Resources"
4. Share settings:
   - Click "Share" → "Anyone with the link"
   - Set to "Viewer"
5. **Copy the folder link** (looks like: https://drive.google.com/drive/folders/XXXXX)
6. Share this link with me

### Create Folder for Student Materials

1. Create another folder: "Elevare Student Materials"
2. Share settings: "Anyone with the link" → "Viewer"
3. Copy link and share with me
